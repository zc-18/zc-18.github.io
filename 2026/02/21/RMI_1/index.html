дёҖгҖҒRMI

1.з®ҖеҚ•дәҶи§Ј

**RMIпјҲRemote Method InvocationпјҢиҝңзЁӢж–№жі•и°ғз”Ёпјү**жҳҜ Java жҸҗдҫӣзҡ„дёҖз§Қж ёеҝғжңәеҲ¶пјҢе®ғе…Ғи®ёиҝҗиЎҢеңЁдёҖдёӘ Java иҷҡжӢҹжңәпјҲJVMпјүдёҠзҡ„еҜ№иұЎпјҢи°ғз”ЁиҝҗиЎҢеңЁеҸҰдёҖдёӘ JVM дёҠзҡ„еҜ№иұЎзҡ„ж–№жі•пјҢе°ұеғҸи°ғз”Ёжң¬ең°еҜ№иұЎдёҖж ·

з®ҖеҚ•жқҘиҜҙпјҢе®ғзҡ„зӣ®зҡ„жҳҜи®©еҲҶеёғејҸ Java еә”з”Ёд№Ӣй—ҙзҡ„йҖҡдҝЎеҸҳеҫ—еҜ№ејҖеҸ‘иҖ…йҖҸжҳҺ

RMI зҡ„ж ёеҝғиҝҗиЎҢжңәеҲ¶

RMI зҡ„еә•еұӮйҖҡдҝЎдҫқиө–дәҺзҪ‘з»ңпјҲйҖҡеёёжҳҜ TCP/IPпјүпјҢе№¶дё”йҮҚеәҰдҫқиө– **Java еәҸеҲ—еҢ–пјҲSerializationпјү**жңәеҲ¶жқҘеңЁзҪ‘з»ңдёӯдј иҫ“еҜ№иұЎгҖӮе®ғзҡ„ж ҮеҮҶе·ҘдҪңжөҒзЁӢеҰӮдёӢпјҡ

  1. RMI RegistryпјҲжіЁеҶҢдёӯеҝғпјүпјҡ жңҚеҠЎз«ҜеҗҜеҠЁж—¶пјҢдјҡе°ҶиҮӘе·ұжҸҗдҫӣзҡ„иҝңзЁӢеҜ№иұЎжіЁеҶҢеҲ°дёҖдёӘеҸ«еҒҡ RMI Registry зҡ„ең°ж–№пјҢе№¶з»‘е®ҡдёҖдёӘеҗҚеӯ—пјҲзұ»дјјз”өиҜқз°ҝпјү
  2. StubпјҲе®ўжҲ·з«Ҝеӯҳж №пјүпјҡ е®ўжҲ·з«ҜйҖҡиҝҮеҗҚеӯ—еҺ»жіЁеҶҢдёӯеҝғжҹҘжүҫпјҢжӢҝеҲ°дёҖдёӘ Stub еҜ№иұЎгҖӮиҝҷдёӘ Stub е°ұеғҸжҳҜжңҚеҠЎз«ҜеҜ№иұЎеңЁе®ўжҲ·з«Ҝзҡ„вҖңд»ЈзҗҶвҖқ
  3. зҪ‘з»ңдј иҫ“пјҡ еҪ“е®ўжҲ·з«Ҝи°ғз”Ё Stub зҡ„ж–№жі•ж—¶пјҢStub дјҡжҠҠи°ғз”Ёзҡ„ж–№жі•еҗҚгҖҒеҸӮж•°зӯүдҝЎжҒҜиҝӣиЎҢеәҸеҲ—еҢ–пјҢйҖҡиҝҮзҪ‘з»ңеҸ‘йҖҒз»ҷжңҚеҠЎз«Ҝ
  4. SkeletonпјҲжңҚеҠЎз«ҜйӘЁжһ¶пјүпјҡ жңҚеҠЎз«ҜжҺҘж”¶еҲ°зҪ‘з»ңиҜ·жұӮеҗҺпјҢSkeletonпјҲеңЁиҫғж–°зҡ„ Java зүҲжң¬дёӯе·ІејұеҢ–дёәеҶ…йғЁжңәеҲ¶пјүдјҡеҸҚеәҸеҲ—еҢ–иҝҷдәӣж•°жҚ®пјҢжүҫеҲ°зңҹжӯЈзҡ„жңҚеҠЎз«ҜеҜ№иұЎе№¶жү§иЎҢж–№жі•
  5. иҝ”еӣһз»“жһңпјҡ жү§иЎҢе®ҢжҜ•еҗҺпјҢз»“жһңеҶҚж¬Ўиў«еәҸеҲ—еҢ–пјҢеҺҹи·Ҝиҝ”еӣһз»ҷе®ўжҲ·з«Ҝзҡ„ StubпјҢжңҖеҗҺи§ЈеҢ…дәӨз»ҷе®ўжҲ·з«Ҝд»Јз Ғ

  2. е®ўжҲ·з«ҜеҸ‘иө·ж–°иҝһжҺҘпјҡ е®ўжҲ·з«Ҝж №жҚ®жҸҗеҸ–еҮәзҡ„ең°еқҖпјҢдё»еҠЁдёҺ RMI Server е»әз«Ӣз¬¬дәҢж¬Ў TCP иҝһжҺҘ

  4. жңҚеҠЎз«Ҝжү§иЎҢе№¶иҝ”еӣһпјҡ жңҚеҠЎз«ҜжҺҘж”¶еҲ° Call ж¶ҲжҒҜпјҢеҸҚеәҸеҲ—еҢ–еҸӮж•°пјҢзңҹжӯЈжү§иЎҢжң¬ең°зҡ„ sayHello() ж–№жі•пјҢжңҖеҗҺе°Ҷз»“жһңеәҸеҲ—еҢ–пјҢйҖҡиҝҮ ReturnData иҝ”еӣһз»ҷе®ўжҲ·з«Ҝ
